# LeetTrack

LeetTrack is a web application that helps users track and analyze their LeetCode problem-solving journey. It enables users to record solved problems, monitor their progress, visualize performance through analytics, and stay consistent with coding goals.

## Planned Features

- User Authentication (Register, Login, Logout)
- Dashboard with coding statistics
- Add, edit, and delete solved problems
- Search and filter problems
- Personal notes for each problem
- Progress analytics and visualizations
- Coding streak tracking
- Daily and weekly coding goals
- Future AI-powered recommendations
- Dark mode
